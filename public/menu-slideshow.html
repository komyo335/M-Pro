Accessibility WCAG 2.1 AA

Built for keyboard, screen readers, and all users

✓ Semantic HTML

  • <nav aria-label="Main navigation">
  • <button> elements (not divs)
  • aria-current="page" on active item

✓ Keyboard Navigation

  • All items are focusable <button> elements
  • :focus-visible outline on Tab navigation
  • Labels expand on keyboard focus too

✓ Screen Readers

  • Icons marked aria-hidden="true"
  • title attribute as fallback label
  • Visible text labels (no icon-only mystery)

✓ Visual

  • Color is never the only indicator
  • Border + background + color all change
  • Respects prefers-color-scheme

Implementation React 19 + CSS
